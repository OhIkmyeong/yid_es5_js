    스멀스멀 다가오는 FP (함수형 프로그래밍)
    - 좋아지는 하드웨어 성능
    - 좋아지는 컴파일러
    - 함수형 프로그래밍 기술
    - 좋아지는 분산 / 리액티브 환경
    - 동시성 + 병렬성 관련 기술
    - 성공적인 적용 사례와 영향

함수형 프로그래밍은 애플리케이션, 함수의 구성요소, 더 나아가서 언어 자체를 함수처럼 여기도록 만들고, 이러한 함수 개념을 가장 우선순위에 놓는다.
함수형 사고방식은 문제의 해결방법을 동사(함수)들로 구성(조합)하는 것
